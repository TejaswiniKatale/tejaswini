# Tejaswini Katale Portfolio Website

This folder is ready for GitHub Pages.

## Files
- `index.html` — main website file
- `assets/` — images used by the website

## How to upload to GitHub
1. Create a new GitHub repository, for example: `tejaswinikatale.github.io`
2. Upload `index.html` and the full `assets` folder to the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Click **Save**.
6. Your site should be live at:
   `https://YOUR-GITHUB-USERNAME.github.io/`

## Important
Do not rename the `assets` folder unless you also update the image paths inside `index.html`.
